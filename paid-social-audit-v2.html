Paid Social Audit Agent

An agent that audits a Meta Ads account the way a senior buyer would. Give it a campaign export and it inspects the data, maps the spend across the funnel, decides its own next move based on what it finds, diagnoses each problem campaign, and writes a prioritised audit with the quick wins ranked first. The plan, act and observe loop is shown live. It runs as a real Claude agent with a key, or a real local decision loop with none.

What the agent needs

read this first

A campaign, ad set or ad level export from Meta Ads Manager. Two minutes:

  1. In Ads Manager, open the Campaigns tab (or Ad sets / Ads for a deeper audit) and set your date range (last 30 days is a good default).
  2. Set the columns to Performance (or a custom set that includes spend, impressions, link clicks, results and result type).
  3. Click Reports or the export icon and choose Export table data as CSV. Set Breakdown to Day first and the agent reads momentum too.
  4. Drop that CSV in below. Nothing leaves your browser except the calls you choose to send to Anthropic.
